Lessons Learned

Which model is appropriate for my data?

The order in which to set up the model. When should we oversample? At which point does oversampling occur? Should we maintain the 1-0 original balance?

Neural Networks are indeed a black box.

Never trust the description of the data you are downloading. A thorough EDA process can save a lot of time.

More steps do not mean better results.

Run your code through a small sample to prevent running out of memory.
